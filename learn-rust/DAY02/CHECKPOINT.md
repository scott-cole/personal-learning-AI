# DAY02: Checkpoint

**Closed book.** No looking back.

1. What is the difference between an expression and a statement in Rust?
   an expression returns a value, a statement does not

2. You write `fn add(x: i32, y: i32) -> i32 { x + y; }`. Will this compile? Why or why not?
   no, because the semicolon makes it a statement returning (), not the sum

3. What does the `->` symbol mean in a function definition?
   the return type of the function

4. What type does a function return if you omit `-> Type` entirely?
   () aka unit

5. You have `let x = { let a = 2; a * 3 };`. What is `x`?
   6
