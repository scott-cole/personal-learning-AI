# DAY16: Checkpoint

**Closed book.** No looking back.

1. What does `.collect()` do on an iterator?
   consumes the iterator and collects elements into a collection type

2. What does `.filter()` return?
   an iterator containing only elements for which the closure returns true

3. What does `.sum()` do?
   consumes the iterator and adds up all elements

4. What is lazy evaluation in the context of iterators?
   the iterator doesn't compute anything until a consuming method like collect or sum is called

5. What does `.fold(0, |acc, x| acc + x)` compute?
   the sum of all elements, starting from 0
