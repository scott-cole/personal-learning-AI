1. What two crates are typically used for logging in Rust?
   log (the facade) and env_logger (the implementation)

2. What environment variable controls env_logger's output level?
   RUST_LOG

3. What function initializes env_logger?
   env_logger::init()

4. Name the five log levels from least to most severe.
   trace, debug, info, warn, error

5. What function reads an environment variable in Rust?
   env::var("VARIABLE_NAME")
