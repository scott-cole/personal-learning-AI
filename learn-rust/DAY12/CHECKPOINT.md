**Closed book.** No looking back.

1. What keyword defines a trait?
   trait

2. You write `fn f(item: &impl MyTrait)`. What does this mean in terms of trait bounds?
   item can be any type that implements MyTrait

3. What does a default implementation in a trait allow?
   types can use the default without implementing that method themselves

4. What is the `where` clause used for in generics?
   to specify trait bounds more readably when they're complex

5. Can a type implement multiple traits?
   yes
