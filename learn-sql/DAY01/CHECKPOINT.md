# DAY01 — Checkpoint

## Questions

**1. What does `SELECT * FROM employees;` return?**

<details>
<summary>Answer</summary>
All columns and all rows from the `employees` table.
</details>

**2. Which SQL keyword is used to retrieve data from a table?**

<details>
<summary>Answer</summary>
`SELECT`
</details>

**3. TRUE or FALSE: A table row is equivalent to a single record.**

<details>
<summary>Answer</summary>
TRUE
</details>

**4. What does the `SERIAL` data type do in PostgreSQL?**

<details>
<summary>Answer</summary>
It creates an auto-incrementing integer column, typically used for primary keys.
</details>

**5. Write SQL to show only the `name` and `salary` columns from a table called `workers`.**

<details>
<summary>Answer</summary>
```sql
SELECT name, salary FROM workers;
```
</details>
