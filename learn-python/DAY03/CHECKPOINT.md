# DAY03 Checkpoint

## Questions

1. What is the result of `"Python"[::-1]`?
2. True or False: `msg = "Hello"; msg[0] = "J"` will change the string to `"Jello"`.
3. What does `"  spaced out  ".strip()` return?
4. How do you combine the list `["a", "b", "c"]` into the string `"a-b-c"`?
5. What does `f"{7 * 8}"` evaluate to?

<details>
<summary>Answers</summary>

1. `"nohtyP"`
2. False — strings are immutable. It raises a `TypeError`.
3. `"spaced out"`
4. `"-".join(["a", "b", "c"])`
5. `"56"`

</details>
