# DAY01 Checkpoint

## Questions

1. What does `type(3.14)` return?
2. Which of these variable names is invalid in Python: `my_var`, `2fast`, `_count`, `userName`?
3. What is the output of `print(f"Half of {10} is {10/2}")`?
4. True or False: In Python, a variable's type is fixed once assigned and cannot change.
5. What does `bool("")` evaluate to?

<details>
<summary>Answers</summary>

1. `<class 'float'>`
2. `2fast` — starts with a digit
3. `Half of 10 is 5.0`
4. False. Python is dynamically typed; a variable can be reassigned to a different type.
5. `False` — empty strings are falsy.

</details>
